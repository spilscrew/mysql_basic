Прототип базы данных портала https://netflix.com.

В прототипе реализована возможность управлением учётными записями пользователей (accounts), профилями учётных записей
(profiles), подписками учётных записей (subscriptions) и информация о платёжных методах (payment_methods).

Добавлена выборка списка почты, полный номер телефона (включая код страны) и название страны пользователя.

При добавлении новой подписки и платёжного метода срабатывает триггер, который добавляет id подписки и платёжного
метода в учётную запись.

Так-же имеется возможность управлениея контентом (content). Для контента имеется отдельная таблица категорий и
представление с выводом списка названий категорий для каждой позиции контента. Второе представление выводит счетчик
количества "лайков" (content_likes) для каждой позиции контента.

В файле db_data.sql присутствует демо-данные для netflix_db.
